# Chatty

Chat App, Just like WhatsApp

### Run

```bash
npm install
npm run dev
```

### Features:

1. Real time chat between 2 people
2. Authentication
3. Profile
4. Add new friend
5. Register
6. Login

### Web Framework

Next.js https://nextjs.org/

### UI

Styling: Tailwind
Testing: Storybook

### Authentication

NextAuth: https://next-auth.js.org/
Email Confirmation

### Database

Redis
