# Chatty

Chat App, Just like WhatsApp

### Run

```bash
npm install
npm run dev
```

### Features:

- Real time chat between 2 people
- Profile
- Add new friend
- Register
- Login

### Web Framework

Next.js https://nextjs.org/

### UI

- Tailwind
- Storybook

### Authentication

NextAuth https://next-auth.js.org/

### Database

Redis
